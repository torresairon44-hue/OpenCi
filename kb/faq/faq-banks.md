## Question
Which banks and financial partners are supported in OpenCI?

## Answer
OpenCI supports the following banks and financial partners:

TFS, SBF, CBS, BDO, BPI, CSB, ESQ, EWB, FHL, FUSE, HOME CREDIT, HSBC, PNB, RSB, RCBC, PSB, UBP, MANULIFE, MAYA, MBTC, MSB, TALA, AUB

### Common Follow-Up Questions

**Can I work with multiple banks?**
Yes, agents can handle cases from different banks. Select the correct bank code when creating demand letters or processing cases.

**How do I know which bank a case belongs to?**
The bank code is displayed in the case details. Each case is tagged with the originating bank.

**Are there different procedures for different banks?**
The core workflow is the same across banks. Some banks may have additional requirements shown as system alerts during form submission.

> **NOTE:** Bank codes must be selected accurately during case creation. Incorrect bank assignment may delay processing.
