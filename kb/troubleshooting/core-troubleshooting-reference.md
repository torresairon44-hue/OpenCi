## Symptoms
This is a consolidated quick-reference for the most common OpenCI issues. For detailed resolution steps, see the individual troubleshooting guides.

## Probable Causes
Issues generally fall into one of four categories: access/login, app updates, GPS/location, or performance.

## Resolution Steps

### Lark Login and Access Issues
- Password resets are handled by the IT department ONLY
- Do not attempt multiple password resets (causes account lockout)
- Contact IT with your employee ID and issue description
- Each user can be logged in on a maximum of 1 device at a time
- See: `troubleshoot-lark.md` for full guide

### App Update Issues
- Missing update button: Uninstall and reinstall from Google Play Store
- Force Stop: Settings → Apps → ChatAI → Force Stop
- Clear Play Store cache: Settings → Apps → Google Play Store → Storage → Clear Cache
- Ensure at least 100 MB free storage space
- See: `troubleshoot-app-update.md` for full guide

### GPS and Location Issues
- Check internet connection (WiFi or mobile data)
- Verify location permissions: Settings → Apps → ChatAI → Permissions → Location → "Always Allow"
- Ensure GPS toggle is ON: Settings → Location
- Go outdoors for better GPS signal; wait 2-3 minutes for initial lock
- Restart location services if needed
- See: `troubleshoot-gps-location.md` for full guide

### Performance Issues
- Slow app response or system lag: Contact Mr. Brann
- Form loading delays: Refresh app or clear cache (Settings → Storage → Clear App Cache)
- If issues persist after cache clear, uninstall and reinstall the app

### Form Submission Errors
- Check all required fields marked with * (red asterisk)
- Verify file formats match supported types (PDF, JPEG, PNG, etc.)
- File size must be under 10 MB
- Check internet connection (WiFi recommended for uploads)
- See: `troubleshoot-forms.md` for full guide

## Escalation Path
1. **Self-Service**: Follow the resolution steps above
2. **Mr. Brann**: For Lark/login issues, form submission errors, QR/reference code scanning, and system performance
3. **IT Department**: For password resets and account lockouts (available 24/7 via IT Hotline)
4. **Area Coordinator**: For GPS visibility issues preventing mandatory tagging

> **IMPORTANT:** Always provide error codes and screenshots when contacting IT or Mr. Brann for faster resolution.
