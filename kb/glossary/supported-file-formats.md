## Term
Supported File Formats

## Definition
The file types accepted by the OpenCI platform for document uploads, form attachments, and case documentation.

## Context

### Accepted Formats
PDF, JPEG, PNG, JPG, HEIC, HEIF, CSV, XLSX, XLS, DOC, DOCX, TXT, ZIP, RAR

### Upload Constraints
- Maximum file size: 10 MB per file
- Files must not be corrupted or password-protected
- Image files should be clear and well-lit for proper review

### Format Categories
- **Documents**: PDF, DOC, DOCX, TXT
- **Spreadsheets**: CSV, XLSX, XLS
- **Images**: JPEG, JPG, PNG, HEIC, HEIF
- **Archives**: ZIP, RAR

> **NOTE:** If a file format is not accepted, try converting images to JPEG or documents to PDF before uploading.
